# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project or on [tohuwabohu.wien](https://tohuwabohu.wien), please **do not** open a public GitHub issue.

Instead, use GitHub's private vulnerability reporting:

👉 [Report a vulnerability](https://github.com/mGallee/tohuwabohu/security/advisories/new)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

## Scope

This policy covers:

- The source code in this repository
- The deployed application at [tohuwabohu.wien](https://tohuwabohu.wien)

## Out of Scope

- Third-party dependencies (please report these upstream)
- Issues in Next.js itself — report to [vercel/next.js](https://github.com/vercel/next.js/security)

## Disclosure Policy

We follow [coordinated disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure). Please give us reasonable time to address the issue before any public disclosure.

Thank you for helping keep tohuwabohu safe. 🙏
